# MoodReader MVP

Monorepo for MoodReader Chrome Extension and Backend Proxy.

## Structure
- `/extension`: Chrome Extension (MV3, Vite, TypeScript)
- `/server`: Backend Proxy (Node.js, Express, TypeScript)

## Local Setup
1. `npm install`
2. `npm run dev:server` (Starts backend on port 8080)
3. `npm run build:ext` (Builds extension to `extension/dist`)
4. Load `extension/dist` as unpacked extension in Chrome.
